Welcome to TitleOnJoin, an innovative Bukkit plugin designed to enrich the Minecraft server experience by greeting players with customizable titles and subtitles upon joining. Boost player engagement with personalized welcome messages and immersive visual effects!

## Features

- **Personalized Welcome Messages**: Display custom titles and subtitles that can include player names or other placeholders.
- **Support for Hex Color Codes**: Enables the use of modern hex color codes for crafting vibrant and striking titles.
- **Conditional Formatting**: Choose between simple color codes and hex color support based on server settings.
- **Dynamic Effects**: Accompany the welcome messages with particle effects and sounds for an impactful player greeting.
- **Configurable Settings**: All messages, effects, and timings can be adjusted in the `config.yml` file to fit your server's needs.

## Installation

1. Download the TitleOnJoin plugin JAR file.
2. Place the JAR file in your server's `plugins` directory.
3. Restart or reload your server.
4. Configure the plugin settings as desired in the `config.yml` file.
5. Enjoy enhancing your server's welcome experience!

## Usage

After installation, customize the title messages, subtitle messages, particle effects, and sounds through the server's `config.yml` file. Use placeholders like `%PlayerName%` to make each message personal and engaging.
